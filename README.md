## What's Here

This repository hosts the open-source word data and query engine behind [Worddit](https://worddit.com/), a **word finder** and **word unscrambler** built for competitive word games. Whether you need a **Scrabble word finder**, a **Words With Friends cheat**, an **anagram solver**, or a **Wordle solver**, this project provides the raw dictionaries and the engine to search them.

## Features

- **Word Finder** — Search by letters, length, starts with, ends in, contains, excludes, and exact positions. Try it: [word-finder](https://worddit.com/word-finder)
- **Word Unscrambler** — Enter a rack of letters and get back all valid plays. Try it: [word-unscrambler](https://worddit.com/word-unscrambler)
- **Anagram Solver** — Rearrange any set of letters into dictionary-valid words. Try it: [anagram-solver](https://worddit.com/anagram-solver)
- **Scrabble Word Finder** — Filter by Scrabble US or UK dictionary with point values. Try it: [scrabble-word-finder](https://worddit.com/scrabble-word-finder)
- **Words With Friends Cheat** — WWF-legal words with game-specific scoring. Try it: [words-with-friends-word-finder](https://worddit.com/words-with-friends-word-finder)
- **Wordle Solver** — Narrow down Wordle answers by green, yellow, and gray clues. Try it: [wordle-solver](https://worddit.com/wordle-solver)
- **Words That Start With** - Search words with prefix. Try it: [words-that-start-with](https://worddit.com/words-that-start-with)
- **Words With Letters** - Search words with letters. Try it: [words-with](https://worddit.com/words-with)
- **Words Ending In** - Search words with suffix. Try it: [words-ending-in](https://worddit.com/words-ending-in)
- **5 Letter Words** — Complete index of five-letter words grouped by start letter, end letter, and letter combinations. Browse: [worddit.com/5-letter-words](https://worddit.com/5-letter-words)

## Why Worddit

Most **word finder** tools give you a single search box and dump results. Worddit gives you layered filtering — length, prefix, suffix, contains, excludes, exact position, and blank/wildcard tiles — all in one workspace. You add one clue at a time, see how the list changes, and never rebuild your search from scratch.

If you play Scrabble, Words With Friends, Wordle, or any word puzzle, the live tools at **[worddit.com](https://worddit.com/)** will save you time and help you find higher-scoring words.
